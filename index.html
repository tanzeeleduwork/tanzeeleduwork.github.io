01 / AI Agent
Agentic Customer
Support System
Groq API · llama-3.3-70b-versatile · Python · Function Calling · JSON Schema

An AI agent that processes raw support tickets through a 4-tool agentic loop — classify urgency, extract structured info, route to the correct team, and draft a professional response. Built with a real while True loop that breaks when the model returns no further tool calls. Within each ticket run, the full message history — tool calls and their results — is fed back into each subsequent LLM call so the model builds on its own prior decisions. Function calling schema enforced on every tool call via Groq's OpenAI-compatible API.

02 / ML
House Price
Prediction Suite
Python · scikit-learn · Random Forest · pandas · Jupyter · Feature Engineering

End-to-end ML pipeline on 19,648 California housing samples. Engineered 3 features — IncomePerOccupant, BedroomRatio, RoomsPerBedroom (note: BedroomRatio and RoomsPerBedroom are reciprocals capturing the same relationship). IncomePerOccupant became the top predictor at 51.9% importance. Removed censored price records capped at the dataset's $500k ceiling to prevent the model learning from artificially clipped values. Final dataset: 7 original + 3 engineered = 10 features.
